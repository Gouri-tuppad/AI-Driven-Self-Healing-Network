# AI-Driven-Self-Healing-Network
AI-driven Django framework for network monitoring, fault detection, congestion analysis, and self-healing management. Provides real-time client tracking, traffic analysis, load balancing recommendations, and centralized network administration.

## Features
- Real-time Client Monitoring
- Network Performance Monitoring
- Fault Detection and Analysis
- Traffic Analysis
- Congestion Detection
- Load Balancing Recommendations
- User Management Dashboard
- AI-Based Self-Healing Mechanisms

## Technologies Used
- Python
- Django
- SQLite/MySQL
- HTML, CSS, JavaScript
- AI/ML Concepts for Fault Detection

## Objective
To improve network reliability, availability, and operational efficiency through intelligent monitoring and automated recovery mechanisms.
